# Chrome ウェブストア掲載情報

## 拡張機能名

Gemini 最新情報スイッチ

## 概要

Geminiに「Webを検索」チェックを追加。必要なときだけ、Webの最新情報を調べるよう促せます。

## 詳細説明

Geminiに、Web検索を。

Geminiで最新のニュース、製品情報、制度、価格などを調べたいときに、Web検索を促すためのChrome拡張機能です。

Geminiの入力エリアに「Webを検索」チェックボックスを追加します。チェックしてプロンプトを送信すると、送信直前に「最新情報」を末尾へ追加し、Web上の新しい情報を踏まえた回答をGeminiに促します。チェックを外しているときは、入力内容を変更しません。

主な機能:

- チェックひとつでGeminiにWeb検索を促す
- 必要なときだけ有効にできる
- ページを再読み込みするとチェックはOFFに戻る
- 開発者サーバーへの通信、広告、アクセス解析なし
- Chrome APIの追加権限なし

本拡張機能はGemini内部の検索設定を直接操作するものではありません。Web検索の実行、回答内容および情報の新しさは、Geminiの仕様、選択モデル、設定および利用状況により異なります。

本拡張機能は独立して開発された非公式ツールであり、Google LLC の提供、承認または提携によるものではありません。

## カテゴリと言語

- カテゴリ: 仕事効率化
- 言語: 日本語
- 対象地域: 指定なし
- 成人向けコンテンツ: なし

## 単一目的の説明

利用者が選択した場合に限り、Geminiへ送信するプロンプトの末尾に「最新情報」を追加し、Web上の最新情報を調べるよう促すこと。

## 権限・ホストアクセスの説明

Geminiの入力欄へ「Webを検索」チェックボックスを表示し、利用者がチェックした場合に送信直前のプロンプトへ文字列を追加するため、`https://gemini.google.com/*` のページ上でのみコンテンツスクリプトを実行します。

Chrome APIの追加権限は要求せず、他のWebサイトでは動作しません。

## リモートコード

「いいえ、リモートコードを使用していません」を選択します。拡張機能に含まれるローカルのJavaScriptとCSSのみを実行します。

## データ利用申告

機能提供のため、利用者がGeminiへ入力したプロンプトを送信時に限ってブラウザ内で一時的に処理します。これはユーザー生成コンテンツに該当します。

- ユーザー生成コンテンツ: ブラウザ内で一時的に処理
- 開発者による収集: なし
- 開発者または第三者サーバーへの独自送信: なし
- 保存: なし
- 広告、トラッキング、アクセス解析: なし
- 販売または目的外利用: なし

処理内容は、プロンプトが空でないこと、末尾に「最新情報」がすでに存在しないことの確認、および必要な場合の文字列追加のみです。利用者が送信した変更後のプロンプトは、通常のGemini利用と同様にGoogleへ送信されます。

Chromeウェブストアの「データ使用」欄では、少なくとも「ユーザー生成コンテンツ」を申告し、すべての限定使用に関する項目へ実態どおり回答します。申告内容は公開するプライバシーポリシーと一致させてください。

## スクリーンショット

Chromeウェブストアへの申請には、実際の拡張機能を表示した1280×800 pxまたは640×400 pxのスクリーンショットが1枚以上必要です。最大5枚まで登録できます。

推奨する構成:

1. Geminiのモデル選択ボタン左側に「Webを検索」が表示されている画面
2. チェックをONにして、Webの最新情報を質問する利用場面
3. チェックOFFでは通常どおり送信できることを示す画面

## 申請時の入力項目

- Store Listing: 概要、詳細説明、カテゴリ、言語、アイコン、スクリーンショット、プロモーション画像
- Privacy: 単一目的、ホストアクセスの理由、リモートコード、データ使用、プライバシーポリシーURL
- Distribution: 公開範囲、対象地域、価格
- Test instructions: Geminiへアクセスし、入力欄の「Webを検索」を確認する手順。ログインが必要な場合はその旨を記載
