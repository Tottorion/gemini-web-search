# Gemini 最新情報スイッチ

GeminiにWeb検索を促すため、入力エリア内のモデル選択ボタン左側へ「Webを検索」チェックボックスを追加するChrome拡張機能です。画面上に重ねて配置するため、チェック欄によって入力欄の高さは変わりません。

チェックした状態でプロンプトを送信すると、送信直前に次の文字列を末尾へ追加します。

```text

最新情報
```

この文字列を追加することで、Web上の最新情報を踏まえた回答をGeminiに促します。Web検索の実行および回答内容はGeminiの仕様、選択モデル、設定および利用状況により異なります。

チェックはページを再読み込みすると OFF に戻ります。開発者サーバーへの外部通信は行わず、Gemini API も使用しません。

## インストール

1. Chrome で `chrome://extensions/` を開きます。
2. 右上の「デベロッパー モード」を有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
4. この `Gemini Web Search` フォルダを選択します。
5. `https://gemini.google.com/` を開くか、すでに開いている場合は再読み込みします。

## 使い方

1. Gemini のモデル選択ボタンの左にある「Webを検索」をチェックします。
2. 通常どおりプロンプトを入力して送信します。
3. チェック中は、Enter キーまたは送信ボタンで送信する直前に `最新情報` が追加されます。

チェックを外して送信した場合、入力内容は変更されません。
