# Gemini 最新情報スイッチ プライバシーポリシー

最終更新日: 2026年7月22日

## 1. 適用範囲

本プライバシーポリシーは、Chrome拡張機能「Gemini 最新情報スイッチ」（以下「本拡張機能」）に適用されます。

## 2. 本拡張機能の目的と動作

本拡張機能は、GeminiにWeb上の最新情報を調べるよう促すことを単一の目的としています。

本拡張機能は `https://gemini.google.com/*` 上でのみ動作します。Geminiの入力エリアに「Webを検索」チェックボックスを表示し、利用者がチェックした状態で送信した場合に、プロンプト末尾へ「最新情報」という文字列を追加します。

本拡張機能はGemini内部の検索設定を直接操作するものではありません。Web検索の実行および回答内容はGeminiの仕様、選択モデル、設定および利用状況により異なります。

## 3. ブラウザ内で処理する情報

本拡張機能は、機能提供のため次の情報を利用者のブラウザ内で一時的に処理します。

- 利用者がGeminiの入力欄へ入力したプロンプト
- 送信操作およびチェックボックスの選択状態
- 入力欄、送信ボタン、モデル選択ボタンなど、機能の表示と実行に必要なGemini画面上の要素

プロンプトは、空でないこと、末尾に「最新情報」がすでに存在しないことの確認、および必要な場合の文字列追加にのみ使用します。

## 4. 開発者による情報の収集

本拡張機能の開発者は、個人情報、認証情報、閲覧履歴、プロンプト内容、Geminiの回答内容、利用状況その他の利用者データを収集しません。

本拡張機能には、開発者が運営するデータ収集サーバー、広告、トラッキングまたはアクセス解析機能はありません。

## 5. 保存および保持期間

本拡張機能は、利用者データをCookie、LocalStorage、`chrome.storage`、ファイル、データベースその他の仕組みに保存しません。ブラウザ内での一時的な処理後、本拡張機能がデータを保持することはありません。

チェックボックスの選択状態も保存せず、ページを再読み込みするとOFFに戻ります。

## 6. 外部送信および第三者提供

本拡張機能は、利用者データを開発者のサーバー、広告事業者、データブローカーその他の第三者へ独自に送信または提供しません。利用者データを販売することもありません。

利用者がGemini上で送信操作を行った場合、変更後のプロンプトは通常のGemini利用と同様にGoogleへ送信されます。この送信は利用者の操作に基づくGeminiの機能によるものです。Googleにおける情報の取扱いについては、Googleが提供する規約およびプライバシーポリシーをご確認ください。

## 7. 権限

本拡張機能はChrome APIの追加権限を要求しません。機能提供に必要な範囲として、`https://gemini.google.com/*` に限定してコンテンツスクリプトを実行し、他のWebサイトでは動作しません。

## 8. Chrome Web Store User Data Policyへの準拠

本拡張機能による利用者データの取扱いは、Chrome Web Store User Data Policy（Limited Use要件を含みます）に準拠し、本拡張機能の単一目的を提供するために必要な範囲に限定されます。

利用者データを、パーソナライズ広告、リターゲティング広告、信用評価、融資目的その他の目的外利用へ使用しません。また、開発者または第三者の担当者が利用者のプロンプトを閲覧することはありません。

## 9. 第三者との関係

本拡張機能は独立して開発された非公式ツールであり、Google LLC の提供、承認または提携によるものではありません。GeminiおよびGoogleはGoogle LLCの商標または登録商標です。

## 10. 本ポリシーの変更

本拡張機能の機能またはデータ取扱いに変更がある場合、本ポリシーを更新し、最終更新日を改定します。
